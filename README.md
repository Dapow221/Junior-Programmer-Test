## Junior Test Programmer

## How to run server-side
- npm install
- nodemon app.js
  
## How to run client-side

- cd junior-test
- npm install
- npm run dev
  
## Description 
Server-side menggunakan Database PostgresSQL Framework Backend Sequelize, Express.js.


Client-side menggunakan Framework Vue.js
